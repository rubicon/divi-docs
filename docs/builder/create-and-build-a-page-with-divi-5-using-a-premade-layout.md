# Create and Build a Page with Divi 5 using a Premade Layout

Start building with Divi's professionally designed layouts. Browse the layout library, import a template, and customize it to match your brand.

## Overview

Divi includes hundreds of premade layouts created by Elegant Themes designers. These layouts save hours of design time—they're professionally styled, fully responsive, and built with industry best practices. Use them as starting points for your pages and customize colors, text, and images to match your brand.

Premade layouts are faster than building from scratch, but you still have complete control to modify every aspect of the design.

## How Premade Layouts Work

A premade layout contains:

- **Sections** – Main horizontal slices of the page (header, hero, features, call-to-action, footer, etc.)
- **Rows and Columns** – Structure that controls the grid layout within sections
- **Modules** – Text, images, buttons, forms, galleries, testimonials, and more
- **Styling** – Colors, fonts, spacing, and responsive rules already applied

You can modify any element while keeping others as-is, or redesign the entire layout.

## Step 1: Create a New Page

1. In the WordPress admin, go to **Pages → Add New**
2. Enter a **page title** (for example, "Home," "About," or "Services")
3. Look for the **Divi Builder options**—you'll see three buttons:
   - **Build from Scratch**
   - **Choose A Premade Layout**
   - **Build with AI**
4. Click **Choose A Premade Layout**

## Step 2: Browse the Layout Library

### Accessing Premade Layouts

After clicking "Choose A Premade Layout," the layout library opens showing available options:

- **Premade Layouts** (by Elegant Themes) – The main library
- **Your Saved Layouts** – Layouts you've created and saved
- **Your Existing Pages** – Pages you've already built (you can ignore these for now)

### Finding the Right Layout

1. The layout packs are organized by type (landing, business, portfolio, blog, etc.)
2. Click a **layout pack** to open it
3. Browse the available page layouts in the pack
4. Each layout shows a **preview thumbnail**

### Choosing a Layout

1. Find the **page layout** you want to use (for example, a "Landing" page layout)
2. Click on it or hover to see a larger preview
3. When you've found the one you want, click **Use This Layout**

The layout loads into the visual editor, ready to customize.

## Step 3: Customize the Layout

### Understanding the Structure

The imported layout contains:

- **Sections** – Top-level containers (entire horizontal bands)
- **Rows and Columns** – Sub-structure within sections (grid layout)
- **Modules** – Individual content elements like text blocks, images, or buttons

### Customizing Content

#### Edit Text

1. Click on any **text module** or **heading module** to select it
2. Click the **gear icon** to open its settings
3. Go to the **Content** tab
4. Edit the text content directly
5. Close the settings panel

#### Change Links

1. Click a **button module**
2. Click the **gear icon** to open settings
3. Find **Button Text** – change the button label
4. Find **Button Link** – change the destination URL (another page on your site or an external URL)
5. Close the settings

#### Replace Images

1. Click an **image module**
2. Click the **gear icon** to open settings
3. Find the image field
4. Click the image to replace it with a new one from your media library
5. Adjust image size and alignment if needed
6. Close the settings

### Reorganizing Sections

1. Use the **Layers panel** (click the Layers icon in the left sidebar) to see the page structure
2. Click and drag sections to **reorder** them
3. Or click the **section handles** on the left edge of each section and drag up or down
4. Drop the section in its new position

### Removing Unwanted Sections

1. Click a **section** to select it
2. Click the **trash icon** that appears (on the section or in the Layers panel)
3. Confirm the deletion
4. The section is removed

Similarly, you can delete rows, columns, or individual modules.

### Duplicating Useful Sections

1. Click a **section, row, or module** to select it
2. Click the **copy/duplicate icon**
3. A copy appears below the original
4. Edit the copy as needed

This is useful when you want similar sections with different content.

## Step 4: Design and Style

### Change Colors, Fonts, and Spacing

1. Click any **element** (section, module, text, etc.)
2. Click the **Design tab** in the settings panel
3. Adjust:
   - **Background colors** – Change section backgrounds
   - **Font styles** – Modify text color, size, and weight
   - **Spacing** – Adjust padding and margins
   - **Effects** – Add shadows, borders, or animations
4. Changes apply in real-time to the preview

### Global Design System

For consistency across your site:

1. First, create a **basic design system** (colors, fonts, buttons) using global Divi variables
2. Link modules to these variables using dynamic content
3. When you customize premade layouts, they automatically inherit your brand system
4. Updating one variable updates all modules using it

## Step 5: Publish

### Save Your Work

1. Click the **Publish** or **Save** button in Divi's top toolbar
2. The page is published (or saved as a draft if you prefer)

### Preview Before Publishing

1. Click the **Exit** button in Divi's top toolbar to exit the builder
2. View the page as a regular visitor would
3. Test links, forms, and responsive layout
4. Return to the builder to make changes if needed

## Step 6: Add to Navigation (Optional)

If you want the page to appear in your site's menu:

1. Go to **Appearance → Menus** in the WordPress admin
2. Select your menu
3. Add the new page to the menu
4. Save the menu

## Step 7: Set as Homepage (Optional)

If this is your home page:

1. Go to **Settings → Reading** in the WordPress admin
2. Select "A static page"
3. Choose your new page from the dropdown
4. Save changes

## Best Practices for Premade Layouts

### Use Consistent Layout Packs

Choose **one layout pack** for your main site pages (Home, About, Services, Contact) so everything shares the same style and visual language.

### Establish Design System First

Before customizing layouts:

1. Define your **brand colors**
2. Choose your **fonts**
3. Create **global Divi variables** for these values
4. Save **common button and section styles** as presets

This makes customizing premade layouts much faster and ensures consistency.

### Don't Over-Customize

Premade layouts are designed by professionals. You don't need to change everything:

- Keep the overall layout structure
- Update text and images
- Adjust colors to match your brand
- Leave spacing and typography mostly as-is

This respects the designer's intent and usually looks better than heavily customized versions.

### Save Your Own Layouts

After customizing a premade layout:

1. Click the **three-dot menu** in the builder
2. Select **Save As Layout**
3. Give it a name (e.g., "My Home Page Template")
4. Save it to your **Divi Library**

Now you can reuse it on other pages or share it with team members.

### Test on Mobile

Always test your customized layout on:

- **Phone** – Use the device preview in the builder
- **Tablet** – Check responsive breakpoints
- **Real devices** – Test on actual phones and tablets if possible

Adjust spacing, font sizes, or column counts if the layout looks cramped on mobile.

## Common Customization Workflow

1. **Create page and import layout** – Choose a premade layout
2. **Edit all text** – Replace placeholder text with your content
3. **Replace all images** – Upload and insert your images
4. **Change colors** – Update colors to match your brand
5. **Add/remove sections** – Customize the layout structure
6. **Adjust spacing** – Fine-tune padding and margins
7. **Test responsive** – Check on different screen sizes
8. **Save and publish** – Publish the page

## Troubleshooting

### Layout Looks Different Than Preview

Premade layouts are styled with default Divi settings. If your site has custom CSS or a child theme, it may affect the appearance. Check for conflicting styles in your child theme's stylesheet.

### Images Not Showing

- Verify the image file exists in your media library
- Check file permissions on your server
- Try re-uploading the image if it still doesn't appear

### Text Overflow or Formatting Issues

- Adjust the text content (shorten long text if columns are narrow)
- Increase column widths or reduce text size for better fit
- Check responsive settings—text may need adjustment for mobile

### Layout Changes Reset

If your changes disappear:

- Ensure you clicked **Save** or **Publish** before exiting the builder
- Check that you're viewing the same page (not a draft)
- Clear your browser cache
- Try a different browser

## Combining Multiple Layouts

You're not limited to one layout per page:

1. Import a **hero/header layout** as your top section
2. Add a **features layout** below it
3. Add a **testimonials layout**
4. Add a **footer layout** at the bottom

This flexibility lets you mix and match sections from different layouts.
