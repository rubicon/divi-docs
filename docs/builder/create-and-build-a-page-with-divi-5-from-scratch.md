# Create and Build a Page with Divi 5 from Scratch

Step-by-step guide to building a complete page from scratch using the Divi 5 Visual Builder.

## Overview

Building a page with Divi 5 from scratch is straightforward: create a new WordPress page, launch the Visual Builder, and compose your page using Sections, Rows, Columns, and Modules. Divi 5 organizes content hierarchically—each Section contains Rows, Rows contain Columns or Nested Rows, and Columns contain Modules or Nested Modules. You can adjust every element's design and content in real time using the Visual Builder's responsive preview, settings panels, and design tools.

This guide walks through creating a simple three-section marketing homepage with a hero, services section, and call-to-action footer.

*The Divi 5 Visual Builder shows sections, rows, and modules in a live canvas with responsive preview.*

## Steps

### Step 1: Create a New Page

1. Go to the WordPress admin dashboard
2. Click **Pages → Add New**
3. Enter a title for your page (e.g., "Home" or "About")
4. In the **Pages** settings sidebar (right side), choose a template if needed—for example, a blank template that removes the header and footer if you want total design control. Divi supports blank-page layouts for custom designs.

![WordPress page creation screen](../assets/screenshots/builder/create-page-from-scratch/step-1-create-page.png){ loading=lazy }
*Create a new page in WordPress with an optional template selection.*

### Step 2: Launch the Divi Builder

1. Click the blue **Use Divi Builder** button
2. A dialog appears with three options:
   - **Build From Scratch** — start with a blank canvas
   - **Choose a Premade Layout** — load a pre-designed layout pack
   - **Build with AI** — let Divi AI generate a layout based on a prompt
3. Select **Build From Scratch** to learn the basics

![Divi Builder launch dialog](../assets/screenshots/builder/create-page-from-scratch/step-2-launch-builder.png){ loading=lazy }
*Choose how you want to start building: blank canvas, premade layout, or AI generation.*

### Step 3: Add Your First Section

1. Click the blue **Add New Section** button near the top of the builder canvas
2. The Section Selection panel opens
3. Choose a **one-column section** to start simple

A Section is a large horizontal block that groups content (hero, services, testimonials, footer, etc.). This first section will be your hero.

![Section selection panel](../assets/screenshots/builder/create-page-from-scratch/step-3-add-section.png){ loading=lazy }
*Add a new Section—the top-level organizational block in Divi.*

### Step 4: Add a Heading Module

1. Hover over the empty Row inside the Section
2. Click the **+** (plus) icon to add a new module
3. In the Module List (which opens automatically), search for and click the **Heading** module
4. In the **Content** tab, add a heading (H1) like "Welcome to Our Website"
5. Switch to the **Design** tab to adjust font size, alignment, and spacing if needed

![Heading module in Visual Builder](../assets/screenshots/builder/create-page-from-scratch/step-4-add-heading.png){ loading=lazy }
*Add a Heading module to your first section and customize the text and design.*

### Step 5: Add a Text Module

1. Hover over the Heading module and click the **+** icon to add a new module
2. Search for and select the **Text** module
3. In the **Content** tab, add your introductory copy
4. Adjust fonts, colors, and spacing in the **Design** tab

![Text module with content](../assets/screenshots/builder/create-page-from-scratch/step-5-add-text.png){ loading=lazy }
*Add descriptive text below the heading.*

### Step 6: Add a Button Module

1. Hover over the Text module and click the **+** icon
2. Add a **Button** module
3. In the **Content** tab, set the button text (e.g., "Learn More")
4. In the **Design** tab, align the button and adjust colors to match your design system

![Button module styled and positioned](../assets/screenshots/builder/create-page-from-scratch/step-6-add-button.png){ loading=lazy }
*Add a call-to-action button styled to match your brand.*

### Step 7: Add a Second Section (Services)

1. Near the bottom of the page, click the blue **Add New Section** button again
2. This time, select a **two-column row**
3. In the left column, add a **Text** module with a heading like "What We Do" and a few short paragraphs
4. In the right column, add an **Image** module with a relevant image
5. Use the **Design** tab to set column spacing, padding, and any background colors

![Two-column section with text and image](../assets/screenshots/builder/create-page-from-scratch/step-7-add-services.png){ loading=lazy }
*A two-column layout for content and imagery—efficient use of space.*

### Step 8: Add a Third Section (Call-to-Action)

1. At the bottom, click **Add New Section** once more
2. Choose a **one-column row**
3. Add a **Text** module for a short headline (e.g., "Ready to work with us?")
4. Below it, add a **Button** module linking to a contact or booking page
5. In the **Design** tab, set a background color to your primary accent color to make it stand out

![CTA section with headline and button](../assets/screenshots/builder/create-page-from-scratch/step-8-add-cta.png){ loading=lazy }
*A dedicated call-to-action section to drive conversions.*

### Step 9: Use the Layers Panel for Quick Navigation

1. Open the **Layers panel** (icon on the left sidebar) to see Sections, Rows, Columns, and Modules in a tidy hierarchical list
2. Click any element in the Layers panel to select it quickly without clicking on the canvas
3. This is especially useful for large pages with many nested elements

![Layers panel showing page structure](../assets/screenshots/builder/create-page-from-scratch/step-9-layers-panel.png){ loading=lazy }
*The Layers panel provides a bird's-eye view of your page structure.*

### Step 10: Preview Different Screen Sizes

1. Look at the **top toolbar** of the builder
2. Use the icons to **Switch between Desktop / Tablet / Mobile** previews and make changes to those specific breakpoints
3. You can also **Zoom in/out** to see more or less of the page
4. Use **Wireframe view** for a simplified structural view without styling

![Responsive preview controls in the toolbar](../assets/screenshots/builder/create-page-from-scratch/step-10-responsive-preview.png){ loading=lazy }
*Test your page design across all breakpoints: desktop, tablet, and mobile.*

### Step 11: Undo, Redo, and History

1. If you make a mistake, use the **Undo / Redo / History** buttons in the toolbar
2. You can quickly undo mistakes or jump back to a previous state using the history panel
3. This lets you experiment without fear

### Step 12: Publish the Page

1. Click the **Publish** button in the Divi top bar—this saves the layout to the page
2. Click **Exit Visual Builder** (also in the top admin bar) to view the page as a visitor would see it
3. In the WordPress editor (if needed), click **Publish** (for a new page) or **Update** (for edits) to finalize everything in WordPress

![Publish button in the Visual Builder](../assets/screenshots/builder/create-page-from-scratch/step-12-publish.png){ loading=lazy }
*Save and publish your page to make it live.*

## Tips

- **Start simple** — use "Build From Scratch" to learn the basics, then explore premade layouts, presets, and Design Variables to speed up future pages
- **Settings organization** — each module and container has three main tabs:
  - **Content** — text, images, links, and basic options
  - **Design** — fonts, colors, spacing, borders, shadows, etc.
  - **Advanced** — custom CSS, conditions, visibility, scroll effects, and more
- **Responsive design** — always check your page on tablet and mobile views before publishing
- **Use the Layers panel** — for complex pages with many nested elements, the Layers panel is faster than clicking on the canvas
- **Save often** — the builder auto-saves, but publish frequently to protect your work

## Common Patterns

### Hero Section
A full-width section with a large headline, subheading, background image or video, and prominent call-to-action button. Best for landing pages and homepages.

![Hero section pattern](../assets/screenshots/builder/create-page-from-scratch/pattern-hero-cta.png){ loading=lazy }
*A classic hero section: image background, text overlay, and primary CTA.*

### Services Grid
A multi-column layout showcasing three or more service cards with icons, titles, and descriptions. Each card is a Blurb or Text module in its own column.

![Services grid pattern](../assets/screenshots/builder/create-page-from-scratch/pattern-services-grid.png){ loading=lazy }
*Three-column services grid with consistent card styling.*

### Testimonials Section
Text or Blurb modules arranged in columns, each containing a quote, author name, and role. Often paired with background colors or subtle borders to stand out.

![Testimonials section pattern](../assets/screenshots/builder/create-page-from-scratch/pattern-testimonials.png){ loading=lazy }
*A multi-column testimonials layout for social proof.*
