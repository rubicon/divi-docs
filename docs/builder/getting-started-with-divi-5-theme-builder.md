## Settings & Options

### General

| Setting | Type | Description |
|---------|------|-------------|
| Log into WordPress Dashboard |  | Ensure you have Divi 5 installed and activated. |
| Navigate to Theme Builder |  | In the WordPress dashboard, go toDivi → Theme Builder. |
| Explore the Interface - |  | You'll see the main Theme Builder screen where all templates are managed. |
| Default Website Template |  | The main template affects all pages unless customized.​Note: This template cannot be deleted. |
| Global Areas |  | Sections forHeader,Body, andFooterthat apply site-wide.​Note: These sections can also be non-global and attached to different templates. |
| Custom Templates |  | Templates assigned to specific pages, posts, or categories. |
| Template Management Tools |  | Options to add, edit, duplicate, or delete templates. |

### Design Tab

| Setting | Type | Description |
|---------|------|-------------|
| Global Templates |  | Create headers, footers, and layouts that apply across your site. |
| Custom Templates |  | Assign specific designs to individual pages, posts, or categories. |
| Drag-and-Drop Interface |  | Organize and manage templates easily. |
| Dynamic Content Integration |  | Add post titles, featured images, and other dynamic elements. |
| Template Import/Export |  | Share and reuse templates between different sites. |
