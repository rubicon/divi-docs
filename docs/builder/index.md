# Divi Builder

The Divi Builder is the visual page building system at the core of Divi. It includes two interfaces: the front-end Visual Builder and the back-end Theme Builder for templates.

## Sections

| Section | Description | Status |
|---------|-------------|--------|

| [Theme Builder](theme-builder.md) | Template system for headers, footers, post templates | 🔲 Needed |
| Layouts Library | Saving and loading layouts | 🔲 Needed |
| Global Elements | Reusable modules and sections | 🔲 Needed |
