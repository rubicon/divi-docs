# Keyboard Shortcuts

A complete reference of keyboard shortcuts available in the Divi 5 Visual Builder.

!!! abstract "Quick Reference"
    **What it does:** Lists all hotkeys for saving, copying, pasting, duplicating, deleting, and navigating in the builder.
    **Where to find it:** Press **?** inside the Visual Builder, or click Help in the left sidebar → Keyboard Shortcuts tab.
    **Key features:**

    - Page-level shortcuts (save, undo, redo, zoom, exit)
    - Element shortcuts (copy, cut, paste, duplicate, delete)
    - Attribute copy/paste shortcuts (Cmd/Ctrl + Shift + C/V)
    - Structure shortcuts for rapid row and section creation (R+1, S+1)

    **ET Docs:** [Divi Keyboard Shortcuts](https://help.elegantthemes.com/en/articles/12992094){:target="_blank"}

## Overview

The Divi Visual Builder includes a comprehensive set of keyboard shortcuts that let you save, undo, copy, paste, duplicate, delete, and navigate without touching the mouse. Learning these shortcuts can dramatically speed up your design workflow. For the official Elegant Themes reference, see [Divi Keyboard Shortcuts](https://help.elegantthemes.com/en/articles/12992094){:target="_blank"}.

You can view the full shortcut list at any time by pressing **?** (or **Shift + ?**) inside the builder, or by clicking the **Help** icon in the left sidebar and selecting the Keyboard Shortcuts tab.

## Page-Level Shortcuts

These shortcuts apply globally while the Visual Builder is active.

| Action | Mac | Windows |
|--------|-----|---------|
| **Save** | Cmd + S | Ctrl + S |
| **Save as Draft** | Cmd + Shift + S | Ctrl + Shift + S |
| **Undo** | Cmd + Z | Ctrl + Z |
| **Redo** | Cmd + Shift + Z | Ctrl + Y |
| **Exit Visual Builder** | Cmd + E | Ctrl + E |
| **Zoom In** | Cmd + + | Ctrl + + |
| **Zoom Out** | Cmd + - | Ctrl + - |
| **Open Page Settings** | O | O |
| **Open Portability (Import/Export)** | P | P |
| **Show Keyboard Shortcuts** | ? or Shift + ? | ? or Shift + ? |

## Element Shortcuts

These shortcuts act on the currently selected element (module, row, section, or column).

| Action | Mac | Windows |
|--------|-----|---------|
| **Copy Element** | Cmd + C | Ctrl + C |
| **Cut Element** | Cmd + X | Ctrl + X |
| **Paste Element** | Cmd + V | Ctrl + V |
| **Duplicate Element** | Cmd + Shift + D | Ctrl + Shift + D |
| **Delete Element** | Cmd + Backspace | Ctrl + Backspace |

## Attribute Copy & Paste Shortcuts

These shortcuts let you transfer design settings between elements without cloning the entire element. See [Copy & Paste Attributes](copy-paste-attributes.md) for full details.

| Action | Mac | Windows |
|--------|-----|---------|
| **Copy Attributes** | Cmd + Shift + C | Ctrl + Shift + C |
| **Paste All Attributes** | Cmd + Shift + V | Ctrl + Shift + V |
| **Paste Design Attributes** | Option + Cmd + V | Alt + Ctrl + V |

## Structure Shortcuts

Quickly add rows and sections or change column layouts with single-key combos.

| Action | Keys |
|--------|------|
| **Add Row (1 column)** | R + 1 |
| **Add Row (2 columns)** | R + 2 |
| **Add Row (3 columns)** | R + 3 |
| **Add Row (4 columns)** | R + 4 |
| **Add Fullwidth Row** | R + F |
| **Add Section with 1 Row** | S + 1 |
| **Change Active Row to 1 Column** | C + 1 |
| **Change Active Row to 2 Columns** | C + 2 |
| **Change Active Row to 3 Columns** | C + 3 |
| **Change Active Row to 4 Columns** | C + 4 |

## Inline Text Editing

| Action | Keys |
|--------|------|
| **Exit Inline Editing** | Esc |

## Tips for Using Shortcuts

- **Start with the basics**: Cmd/Ctrl + S (save) and Cmd/Ctrl + Z (undo) are the two shortcuts you should commit to muscle memory first.
- **Use attribute paste shortcuts** to quickly replicate styling across elements without opening the [right-click menu](right-click-menus.md).
- **Structure shortcuts** (R + 1, S + 1, etc.) are particularly useful when building page layouts from scratch, as they eliminate the need to click through the add-element UI.
- **Combine with the Command Center** for an almost entirely keyboard-driven workflow.
