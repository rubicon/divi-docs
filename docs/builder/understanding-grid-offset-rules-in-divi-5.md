For example:5: Selects the fifth item in the grid.3n: Selects every 3rd element in the grid (item 3, item 6, item 9, etc.)3n+1: Selects grid items 1, 4, 7, 10, etc.3n+2: Selects grid items 2, 5, 8, 11, etc.-n+3: Select the first 3 grid items (item 1, item 2, and item 3) |
