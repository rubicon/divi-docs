# The Instagram Feed Module in Divi 5

The Divi 5 **Instagram Feed Module** pulls images from an Instagram account and renders them as a gallery, with the same image **aspect ratio** and **framing** controls Divi 5 ships across the rest of the builder.

## Overview

<!-- TODO: Add ET help article source_url to frontmatter and expand once network access permits a full scrape. -->

This module was introduced as part of the **Five New Modules for Divi 5** release (Timeline, Breadcrumbs, Table of Contents, SVG, Instagram Feed). Per the announcement, the feed inherits Divi 5's new image aspect ratio and framing settings, so an Instagram gallery can be normalized into a consistent grid regardless of the source posts' aspect ratios.
