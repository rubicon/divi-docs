# The Table of Contents Module in Divi 5

The Divi 5 **Table of Contents Module** automatically generates a clickable table of contents from the headings on a page or post — well suited to Theme Builder post templates where you want long-form content to expose its own outline.

## Overview

<!-- TODO: Add ET help article source_url to frontmatter and expand once network access permits a full scrape. -->

This module was introduced as part of the **Five New Modules for Divi 5** release (Timeline, Breadcrumbs, Table of Contents, SVG, Instagram Feed). It is most useful inside Theme Builder templates so every long-form post inherits an outline automatically rather than requiring per-post markup.
