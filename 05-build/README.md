# 05-build — What Goes Here

Build-phase artifacts. Populated once the project moves past discovery.

## Expected Files

- **`sitemap.md`** — Final approved site architecture
- **`wireframes/`** — Design wireframes (image files or links to Figma/other tools)
- **`platform-decision.md`** — Platform / stack decision record, with reasoning, written at the moment the decision is made. Do not pre-fill this file with a recommendation before the user and the client decide together.

## Note on Platform Decisions

Platform choice (WordPress/Divi vs. Webflow vs. something else) is typically deferred until content is finalized. Any Claude agent tempted to recommend a platform before that point should instead capture the considerations in draft form and flag for the user's review.
