# Client Profile — Divi 5 Technical Documentation

## The Project

This repository produces community-maintained technical documentation for Divi 5, including builder workflows, modules, troubleshooting guides, and maintenance scripts that track upstream changes. The operating focus is to turn scattered information into practical, implementation-ready guidance.

**Website (current):** https://16wells.github.io/divi-docs/

**Key metrics / social proof:**
- Public documentation site is live and accessible.
- Repository includes automated workflows for publishing and monitoring updates.
- Project contains broad topic coverage across builder, modules, and troubleshooting.
