# Contributing to Divi Community Docs

See the full contributing guide on the docs site: [Contributing](https://16wells.github.io/divi-docs/contributing/)

## Quick Start

1. Every page has an edit button — click it to propose changes in your browser
2. For new pages, copy `templates/doc-template.md` and follow the format
3. Submit a pull request and we'll review it

## Local Development

```bash
# Clone the repo
git clone https://github.com/16wells/divi-docs.git
cd divi-docs

# Install MkDocs Material
pip install mkdocs-material

# Run the dev server
mkdocs serve

# Open http://localhost:8000
```
