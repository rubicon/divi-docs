# Elegant Themes blog → divi-docs map

*Purpose:* Track **Divi Resources** and **Theme Releases** posts on `elegantthemes.com` so each one is linked from the **canonical reference page** for that feature. Reference docs stay the source of truth; blog posts are linked as **long-form tutorials** (see `SKILL.md` — *Elegant Themes blog tutorials*).

## How to use this file

1. When Elegant Themes publishes a new Divi 5 tutorial, add a row below (or update the row if the URL slug changes).
2. On the listed **doc page**, ensure there is an **`## Elegant Themes tutorials`** section (see `SKILL.md` for placement and link format). Published docs link to this file on **GitHub** (not a relative path) so `mkdocs build` does not warn — use `https://github.com/16wells/divi-docs/blob/main/planning/et-blog-tutorials-map.md`.
3. During **weekly** maintenance (`claude-code/task-weekly-monitor.md`), skim [Divi Resources](https://www.elegantthemes.com/blog/category/divi-resources) and [Theme Releases](https://www.elegantthemes.com/blog/category/theme-releases) for new posts and update this map + the matching doc pages.
